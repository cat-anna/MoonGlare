# MoonGlare
Simple 3d game engine

## Dependencies
* Boost
* GLEW, GLFW3
* Assimp, FreeImage, FreeType
* BulletPhysics, Eigen3, GLM
* Lua, LuaBridge
* PugiXML
* zLib
* libFormat
* Qt (for editor)

# Some features
* Deffered shadding with spot ligth shadows
* Scripted in lua, configured in xml
* Editor for object/scene configuration
